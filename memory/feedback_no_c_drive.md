Jangan pernah menyimpan file apapun ke Local Disk C (C:\), termasuk download, file output, script sementara, dsb.

**Why:** Preferensi user — C: hanya untuk sistem Windows, semua data/file kerja di drive lain (D: dst).

**How to apply:** Setiap kali akan menyimpan/download/menulis file, pastikan path-nya bukan di C:\. Gunakan D:\ atau path yang user tentukan.
